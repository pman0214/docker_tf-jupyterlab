# TensorFlow + Jupyterlab

## Overview

* Docker image based on ``tensorflow/tensorflow:latest-gpu-py3``.
* Installs python libraries below:
  * keras
  * seaborn
  * jupyter
  * jupyterlab
  * scikit-learn
  * imblearn
* Also provides sample `docker-compose.yml` files.
